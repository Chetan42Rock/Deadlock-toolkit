# Deadlock Detection Toolkit

## 📌 Project Overview

The **Deadlock Detection Toolkit** is a comprehensive web-based application that helps in detecting deadlocks in concurrent systems using various algorithms. It provides an interactive interface with drag-and-drop functionality for specifying resource allocation, visualization of graphs, and detection of cycles indicating deadlocks.

## 🚀 Features
- **Interactive Graph Visualization:** Visual representation of processes and resources with different shapes and colors.
- **Drag-and-Drop Interface:** User-friendly UI for defining resource allocation.
- **Deadlock Detection:** Automatic detection of deadlocks using algorithms and graph traversal.
- **Resource and Process Management:** Add, edit, and delete processes and resources dynamically.
- **Error Handling and Feedback:** User feedback for errors and successful detection of deadlocks.
- **Modes of Input:** Manual input and Drag-and-Drop interface for defining allocations.

## 🔧 Technologies Used
- **Frontend:** React, Bootstrap, React-Vis-Network-Graph
- **Backend:** Flask (Python)
- **Visualization Library:** react-vis-network-graph
- **Version Control:** GitHub

## 📁 Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/YourUsername/Deadlock-toolkit.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Deadlock-toolkit
   ```
3. Install frontend dependencies:
   ```bash
   npm install
   ```
4. Install backend dependencies:
   ```bash
   pip install -r requirements.txt
   ```
5. Run the backend server:
   ```bash
   python app.py
   ```
6. Run the React frontend:
   ```bash
   npm start
   ```
